# stima_bruteforce
Tugas kecil strategi algoritma 1: menyelesaikan permasalahan cryptarithmetic dengan algoritma brute force

## Build: 
```shell
$ g++ ./src/stima1.cpp -o ./bin/stim
```

## Run: 
```shell
$ ./bin/stim <file input>
```
